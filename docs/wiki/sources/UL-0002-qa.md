# Q&A — UL-0002 초안 보완 (개인화 분석 기능의 거취)

- 관련 UL: UL-0002 (온보딩 장벽 해소를 위한 배포 아키텍처 재설계)
- 질문 배경: lint-ul 파이프라인 내부 검사에서, living-spec.md에 기록된 `profile.json` 기반 개인화 분석 기능(보유 기술·프로젝트와 공고 매칭도 비교, 결과를 Notion 페이지에 추가)이 UL-0002 초안의 현재결정사항에서 전혀 언급되지 않아, Notion→Git 저장 전환 이후 이 기능의 존속 여부가 불명확하다는 점이 지적됨.

**Q. profile.json 기반 개인화 분석은 Notion→Git 저장 전환 이후 어떻게 되는가?**

A. Git 저장 방식으로 전환된 이후에도 개인화 분석 기능 자체는 그대로 유지한다. 매칭 결과를 Notion 페이지 블록에 추가하던 것을, 저장 매체가 바뀐 Git 파일(마크다운)에 함께 기록하는 것으로 저장 매체만 대체한다. 기능 자체의 축소·폐기는 없다.
